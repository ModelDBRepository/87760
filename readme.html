Matlab code for the Bruce et al. series of cochlear implant models

This is a public distribution of the code for the models of:

Bruce, I. C., White, M. W., Irlicht, L. S., O'Leary, S. J., Dynes, S., Javel, E., Clark, G. M. (1999). "A stochastic model of the electrically stimulated auditory nerve: Single-pulse response," IEEE Transactions on Biomedical Engineering, 46(6):617-629.

Bruce, I. C., White, M. W., Irlicht, L. S., O'Leary, S. J., Dynes, S., Javel, E., Clark, G. M. (1999). "A stochastic model of the electrically stimulated auditory nerve: Pulse-train response," IEEE Transactions on Biomedical Engineering, 46(6):630-637.

Bruce, I. C., White, M. W., Irlicht, L. S., O'Leary, S. J., and Clark, G. M. (1999). "The effects of stochastic neural activity in a model predicting intensity perception with cochlear implants: Low-rate stimulation," IEEE Transactions on Biomedical Engineering 46(12):1393–1404.

Bruce, I. C., Irlicht, L. S., White, M. W., O'Leary, S. J., and Clark, G. M. (2000). "Renewal-process approximation of a stochastic threshold model for electrical neural stimulation," Journal of Computational Neuroscience 9(2):119–132.

See the file readme.txt for more details. (This readme file is also included in the .zip and .tar.gz archives below.)

The code is archived in the following formats:
  • A PKZIP-compatible .zip file: CImodels.zip
  • A .tar.gz file: CImodels.tar.gz

The files in both of these archives are identical and are suitable for Matlab versions 5.x and greater.
